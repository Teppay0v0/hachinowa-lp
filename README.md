# はちのわ LP（ワイヤーフレーム版）

全国蜜蜂レスキューネットワーク「はちのわ」のランディングページ ワイヤーフレーム版です。

## 公開URL

GitHub Pages で公開中：
https://teppay0v0.github.io/hachinowa-lp/

## 構成

- `index.html` — LP本体（HTML / CSS / JS が1ファイルに収まっています）
- `images/` — 背景イラスト・蜂のSVG
- `wireframe.svg` — 初期段階のワイヤーフレーム

## デザイン方針

- **構成**：中央モバイル幅（460px）＋左右に景色（住宅街・公園・街が時間で切替）
- **中央のすりガラス**：景色がうっすら透けて見える
- **蜂**：すりガラスの向こう側を、ランダムに左右両方向へ飛ぶ。羽根はぱたぱた
- **CTA**：左パネルに「分蜂を報告」「養蜂家として登録」を常時表示
- **配色**：黄色（メイン）＋黄緑（サブ）＋コーラル（POP差し色）
- **フォント**：Cherry Bomb One（英字）＋ Zen Kaku Gothic New（和文）

## 確認したいこと（主宰さん向け）

- 全体の雰囲気
- カラー・トンマナ
- コピー（特に Hero「ハチの大群!? それ、分蜂かも！」）
- 蜂の飛び方
- 「世界の動き」セクション（海外事例）の見せ方
- 各セクションの構成・順番
